# Simple Bank

### How to run (needs Docker)
copy the `.env.example` content into a new file called `.env`

Run the application using Go dependencies locally (requires Go)
```zsh
  make dev-local
```

Run the application with Docker
```zsh
  make dev-docker
```

Run tests
```zsh
  make test
```

See test coverage
```zsh
  make test-coverage
```
